# HIS 商用交付手册（医疗版）

**版本**：1.0 | **细胞**：医院信息系统（HIS）基础商用版 | **行业**：医疗数据合规

## 1. 交付范围

### 1.1 核心行业功能

| 环节 | 说明 | 状态 |
|------|------|------|
| 患者管理 | 患者 CRUD；姓名/身份证加密存储、脱敏展示 | 已实现 |
| 挂号 | POST /registration，幂等 | 已实现 |
| 就诊 | 就诊记录 CRUD，按医生过滤 | 已实现 |
| 处方 | POST /prescriptions，防重复开具 | 已实现 |
| 收费 | POST /charges 幂等；POST /charges/<id>/pay 缴费，欠费提示 | 已实现 |
| 住院（基础） | POST /inpatients | 已实现 |
| 病历（基础） | POST/GET /medical-records，仅追加不可篡改 | 已实现 |

### 1.2 医疗合规与数据安全

- 患者信息（姓名/身份证）加密存储 + 脱敏展示（姓名脱敏、身份证部分隐藏）。
- 医护人员权限分级：医生仅能查看本人负责患者（X-User-Id 过滤患者/就诊列表）。
- 就诊记录日志可审计（人性化审计）。
- 挂号/收费接口幂等；处方防重复；欠费时返回商用化提示。
