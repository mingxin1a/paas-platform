# EMS 商用交付手册（能耗版）

**版本**：1.0 | **细胞**：能耗管理系统（EMS）| **行业**：工业能耗合规

## 1. 交付范围

本手册适用于 EMS 细胞按「行业合规商用可交付」标准的交付验收。EMS 独立部署，仅通过 PaaS 层交互；多租户能耗数据隔离。

### 1.1 核心行业功能

| 环节 | 说明 | 状态 |
|------|------|------|
| 能耗采集 | 模拟采集接口，POST /consumption-records（幂等） | 已实现 |
| 能耗统计 | GET /stats?period=day\|week\|month\|year | 已实现 |
| 能耗分析 | 基于统计维度聚合 | 已实现 |
| 能耗预警 | GET/POST /alerts，异常预警 | 已实现 |
| 能耗报表 | 按日/周/月/年统计报表，符合工业能耗规范 | 已实现 |
| 数据导出 | GET /export?fromDate=&toDate=，符合监管要求 | 已实现 |

### 1.2 行业合规

- **数据留存≥3年**：EMS_ENERGY_DATA_RETENTION_DAYS（默认 1095），GET /config/retention。
- **数据安全**：能耗数据可加密存储（应用层/KMS）；多租户隔离。
- **可靠性**：能耗采集接口幂等（X-Request-ID）；预警可对接实时推送。

## 2. 部署与配置

| 环境变量 | 说明 | 默认 |
|----------|------|------|
| PORT | 服务端口 | 8010 |
| EMS_ENERGY_DATA_RETENTION_DAYS | 能耗数据保留天数 | 1095（3年） |
