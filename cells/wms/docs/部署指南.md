# WMS 细胞 - 部署指南

**版本**：1.0.0

## 1. 部署方式

- **独立运行**：本目录执行 `docker compose up -d` 或 `python src/app.py`，监听 `PORT`（默认 8003）。
- **平台联调**：网关将 `/api/v1/wms/*` 转发至本细胞，需配置 `CELL_WMS_URL`。

## 2. 环境变量

- `PORT`：服务端口。
- `X-Tenant-Id`：由网关传入，多租户隔离。

## 3. 依赖

- Python 3.10+，Flask。当前为内存存储；生产可对接独立数据库（见 `database_schema.sql`）。

## 4. 验证

- `GET {CELL_URL}/health` 返回 200。
- 项目根执行：`./run.sh verify wms` 或 `./scripts/verify_delivery.sh wms`。
