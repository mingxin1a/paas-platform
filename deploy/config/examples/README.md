# deploy/config/examples — 配置示例

本目录集中放置**示例配置**，便于归类与复制使用。

| 文件 | 说明 | 使用方式 |
|------|------|----------|
| `governance_config.example.yaml` | 治理中心配置示例（可选） | 按需复制到 `deploy/` 或合并到实际配置 |

其他示例位置：
- **环境变量**：`deploy/.env.example`（复制为 `deploy/.env`）
- **监控**：`deploy/monitoring/prometheus.example.yml`、`deploy/monitoring/alerts.example.yaml`
