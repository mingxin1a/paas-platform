# glass_house - 玻璃房透明化目录

本目录存放自主进化引擎与**数字创业孵化器**的透明化产出，人类可随时查看。符合《00_最高宪法》修正案 #14（玻璃房原则）。

- **live_evolution_log.md**：实时演化日志，当前引擎状态、细胞快照、周期日志。
- **incubator_manifest.md**：孵化器治理总纲（细胞=独立项目、自发现与孵化、知识进化、自愈）。
- **human_intervention_queue.md**：待人工介入事件队列（严重问题由此进入）。
- **incubator_actions.log.md**：孵化与进化动作记录（外部/内部孵化、对标、自愈）。
- **gap_analysis/**：各细胞的《{CELL_NAME}_gap_analysis.md》对标分析报告。
- **state/**：引擎状态（如 registry.json、last_gap_at）供脚本读取。
- **architecture_decisions.md**：架构决策记录。

所有分析报告、新代码引用、状态变更均在此或通过 live_evolution_log 引用可追溯。
