# 通用贸易企业解决方案 · 功能清单

**模块组合**：CRM + ERP + WMS + SRM + OA  

---

## 一、CRM

| 功能域 | 功能点 | 说明 |
|--------|--------|------|
| 客户 | 客户档案、联系人、跟进记录 | 与商机/合同关联 |
| 商机 | 商机阶段、金额、预测 | 销售漏斗 |
| 合同与回款 | 合同、回款计划与执行 | 与 ERP 应收衔接 |
| 报表 | 销售漏斗、客户分析 | 权限内数据 |

---

## 二、ERP

| 功能域 | 功能点 | 说明 |
|--------|--------|------|
| 采购 | 采购需求、订单、到货 | 与 SRM、WMS 衔接 |
| 销售 | 销售订单、发货、出库 | 与 CRM、WMS 衔接 |
| 库存 | 库存账、批次、成本 | 与 WMS 实物账对账 |
| 财务主线 | 应收应付、成本核算 | 基础闭环 |

---

## 三、WMS

| 功能域 | 功能点 | 说明 |
|--------|--------|------|
| 入库 | 采购入库、退货入库 | 与 ERP 采购衔接 |
| 出库 | 销售出库、调拨 | 与 ERP 销售衔接 |
| 库存 | 库位、批次、盘点 | 与 ERP 库存联动 |

---

## 四、SRM

| 功能域 | 功能点 | 说明 |
|--------|--------|------|
| 供应商 | 档案、准入、分类 | 与采购订单关联 |
| 询报价 | 询价、比价、报价单 | 可选招投标 |
| 订单协同 | 订单下发、确认、交期 | 与 ERP 采购同步 |
| 对账 | 对账单、发票、付款 | 与 ERP 应付衔接 |

---

## 五、OA

| 功能域 | 功能点 | 说明 |
|--------|--------|------|
| 审批 | 请假、报销、合同等审批流 | 流程可视化 |
| 公告与任务 | 公告发布、任务分配与跟踪 | 协同 |
| 电子签章 | 审批单签章（可选） | 合规留痕 |

---

## 六、模块选配说明

- **必选**：CRM、ERP、WMS（贸易核心）。
- **推荐**：SRM（采购透明）、OA（审批与协同）。
- **可选**：TMS（发运），按需增加 Cell。
