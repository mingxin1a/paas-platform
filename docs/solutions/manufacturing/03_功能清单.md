# 通用制造业解决方案 · 功能清单

**模块组合**：ERP + MES + WMS + SRM + TMS  

---

## 一、ERP（企业资源计划）

| 功能域     | 功能点           | 说明                     |
|------------|------------------|--------------------------|
| 物料与 BOM | 物料主数据、BOM  | 支持多阶 BOM、替代料     |
| 计划       | 主计划、MRP、工单 | 计划下达到工单/采购需求  |
| 采购       | 采购需求、订单、到货 | 与 SRM 协同可扩展       |
| 生产       | 工单、领料、报工 | 与 MES 对接执行与反馈   |
| 销售       | 订单、发货、出库 | 与 WMS/TMS 衔接          |
| 库存       | 库存账、批次、成本 | 与 WMS 实物账可对账     |
| 财务主线   | 应收应付、成本核算 | 基础财务闭环             |

---

## 二、MES（制造执行）

| 功能域     | 功能点           | 说明                     |
|------------|------------------|--------------------------|
| 工单管理   | 工单接收、派工、报工 | 与 ERP 工单同步          |
| 工艺与 BOM | 工艺路线、工序、BOM | 支持报工与用料           |
| 生产实绩   | 报工、完工、质量采集 | 可扩展 SPC/质检          |
| 设备与产能 | 设备状态、产能视图 | 可选扩展                 |

---

## 三、WMS（仓储管理）

| 功能域     | 功能点           | 说明                     |
|------------|------------------|--------------------------|
| 入库       | 采购入库、生产入库、退货入库 | 支持上架、库位           |
| 出库       | 销售出库、领料出库、调拨   | 拣配、复核、批次         |
| 库存       | 库位、批次、盘点、先进先出 | 与 ERP 库存账联动        |
| 条码/扫描  | 扫码入库、扫码出库 | 可选 PDA/移动端          |

---

## 四、SRM（供应商关系）

| 功能域     | 功能点           | 说明                     |
|------------|------------------|--------------------------|
| 供应商     | 档案、准入、分类   | 与采购订单关联           |
| 询报价     | 询价、比价、报价单 | 支持招投标流程           |
| 订单协同   | 订单下发、确认、交期 | 与 ERP 采购订单同步      |
| 对账与付款 | 对账单、发票、付款 | 与 ERP 应付衔接          |

---

## 五、TMS（运输管理）

| 功能域     | 功能点           | 说明                     |
|------------|------------------|--------------------------|
| 运单       | 运单创建、承运商、路线 | 与销售出库/发货单关联    |
| 在途       | 轨迹、状态更新     | 支持第三方轨迹接口       |
| 签收       | 签收回单、异常登记 | 客户回签与索赔           |

---

## 六、模块选配说明

- **必选**：ERP（制造企业核心）、WMS（实物与库存一致）。
- **推荐**：MES（计划与执行闭环）、SRM（采购透明）、TMS（发运可视）。
- **可选**：PLM（研发与 BOM）、EMS（能耗），按需增加 Cell 与路由即可。
