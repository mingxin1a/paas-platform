# 细胞独立交付文档索引

各细胞独立交付时，交付包内需包含「细胞独立交付文档」；可复用本仓库已有文档并指向统一规范。

---

## 统一交付包结构

见 **《商用交付包目录结构》**：`cells/<CELL_ID>/` 下含 `src/`、`Dockerfile`、`k8s/`、`database_schema.sql`、`docs/`。

---

## 各细胞文档位置

| 细胞 | 档案/接口/交付文档 |
|------|---------------------|
| CRM | docs/细胞档案/CRM_细胞档案.md；commercial_delivery/细胞/（按需） |
| ERP | docs/细胞档案/ERP_细胞档案.md；cells/erp/docs/ |
| WMS/TMS/MES/OA/SRM/PLM/EMS | docs/细胞档案/；commercial_delivery/细胞/{WMS,TMS,MES,...}/ |
| HIS/LIS/LIMS/HRM | docs/细胞档案/；cells/*/docs/ |

各细胞 `docs/` 下建议包含：

- **细胞独立交付文档.md**：部署步骤、环境变量、依赖、与网关/事件总线对接说明。
- **管理员手册.md**：配置、运维、备份。
- **接口文档**：可引用平台 OpenAPI（docs/api/openapi.yaml）及细胞 path 约定。
