# LIMS 商用交付手册

**版本**：1.0 | **细胞**：实验室信息管理系统（LIMS） | **对象**：交付/运维

## 功能概述

样品、实验任务、实验数据、实验报告、数据溯源；实验数据留存≥5年（配置）；任务/数据/报告创建写 trace 可审计；实验人员权限（X-User-Id 过滤样品/任务）。

## 部署与运维

- 镜像：`cells/lims`；端口 8013；网关 `CELL_LIMS_URL`。健康：`/health`；留存配置：`/config/retention`。

**详细**：见 `cells/lims/docs/LIMS商用交付手册（实验室版）.md`。
