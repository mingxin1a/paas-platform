# ERP 用户操作手册

**版本**：1.0 | **细胞**：ERP | **对象**：财务、采购、生产等最终用户

## 使用入口与权限

- 经客户端登录后进入「企业资源」或「ERP」模块；仅能操作有权限的菜单与数据。
- 所有金额以「元」在界面展示（接口为分），请以系统显示为准。

## 常用操作（概要）

| 模块 | 操作 |
|------|------|
| 总账 | 科目查询、凭证录入、余额与试算平衡查看 |
| 应收/应付 | 发票录入与查询、收款/付款登记、账龄查看 |
| 物料与采购 | 物料档案维护、采购订单创建与查询 |
| 生产 | BOM 维护、工单创建与报工 |

**详细步骤与截图**：见 `cells/erp/docs/ERP用户操作指南.md`。
