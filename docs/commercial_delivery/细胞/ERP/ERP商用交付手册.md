# ERP 商用交付手册

**版本**：1.0 | **细胞**：企业资源计划（ERP） | **对象**：交付/运维

## 功能概述

- **总账（GL）**：科目、凭证、余额、试算平衡。
- **应收/应付（AR/AP）**：发票、收款/付款、账龄。
- **物料与采购（MM）**：物料主数据、采购订单。
- **生产（PP）**：BOM、工单、报工。
- 金额以「分」为单位；支持幂等、审计、软删除。

## 部署与运维

- 镜像与依赖：`cells/erp`；环境变量 `PORT`（默认 8002）、租户等。
- 网关配置 `CELL_ERP_URL`；一键部署：`--only=erp-cell`。
- 健康：`/health`；指标与日志见 cells/erp/docs。

## 监控与备份

- Prometheus 抓取 `/health`；业务指标若有 `/metrics` 可一并配置。
- 数据备份：按实际存储（内存/MySQL）执行备份脚本。

**详细**：见 `cells/erp/docs/ERP商用交付手册.md`、`cells/erp/docs/ERP管理员手册.md`。
