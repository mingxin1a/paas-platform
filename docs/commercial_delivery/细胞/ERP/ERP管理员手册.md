# ERP 管理员手册

**版本**：1.0 | **细胞**：ERP | **对象**：管理员

## 权限与参数

- 平台侧分配用户可访问细胞（allowedCells 含 erp）；ERP 内按租户隔离，无额外角色配置时所有同租户用户可见同租户数据。
- 系统参数：PORT、数据库连接（生产切 MySQL 时）、审计开关等，见 cells/erp 配置。

## 故障处理

| 现象 | 处理 |
|------|------|
| /health 失败 | 检查进程与数据库连接，重启 erp-cell |
| 凭证/订单保存失败 | 查看返回错误码与 message；常见为参数校验或业务规则（如余额不足） |

**详细**：见 `cells/erp/docs/ERP管理员手册.md`。
