# 商用交付包目录结构（统一规范）

**原则**：细胞独立交付；PaaS 核心与各细胞采用统一目录结构。

## 1. PaaS 核心交付包

```
paas-core/
├── src/
├── Dockerfile
├── k8s/ (deployment, service, configmap)
├── init/init.sql (可选)
└── docs/
```

## 2. 细胞统一结构

```
cells/<CELL_ID>/
├── src/
├── Dockerfile
├── k8s/
├── database_schema.sql
├── delivery.package
└── docs/ (细胞独立交付文档、管理员手册、接口文档)
```

## 3. 全平台

```
paas-platform/
├── platform_core/
├── cells/
├── frontend/
├── glass_house/
├── docs/ (运维手册、安全说明、api/openapi.yaml、错误码表)
└── deploy/
```
